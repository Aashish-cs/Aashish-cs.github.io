# Ashish Mishra Portfolio

Static portfolio site for Ashish Mishra.

## Files

- `index.html` - main portfolio page
- `resume.html` - editable resume source
- `assets/Ashish_Mishra_Resume.pdf` - printable resume linked from the portfolio
- `assets/` - favicon, portrait, and project screenshots

## Deploy

Upload the contents of this folder to GitHub Pages, Vercel, Netlify, or any static hosting service.

For GitHub Pages:

1. Create a repository named `Aashish-cs.github.io` or use any repo with Pages enabled.
2. Copy everything inside this `outputs/` folder to the repository root.
3. Commit and push.
4. Enable GitHub Pages from the repository settings if it is not already enabled.

## Update Resume PDF

Edit `resume.html`, then print it to PDF as `assets/Ashish_Mishra_Resume.pdf`.
